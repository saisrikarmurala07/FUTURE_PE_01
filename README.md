# 🚀 AI Website Copy Generator

## 📌 Overview
This project uses AI tools like ChatGPT, Claude, and Gemini to generate high-quality website content for businesses.

It creates:
- Homepage content
- Service page descriptions
- Call-To-Action (CTA) sections

## 🛠 Tools Used
- ChatGPT
- Claude
- Gemini
- Framer AI
- Google Docs / MS Word
- GitHub

## 🎯 Features
- Generates homepage copy with strong value proposition
- Creates service pages tailored to business types
- Writes persuasive CTA sections
- Adapts tone for different businesses
- Produces website-ready content

## 📦 Deliverables
- Structured prompts
- AI-generated outputs
- Documented workflow

## 🔄 Workflow
1. Define business type
2. Create prompt
3. Generate content using AI
4. Refine output
5. Store results

## 📌 Future Scope
- API integration
- UI-based generator
- SEO optimization
